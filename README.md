# üçé Apple Disease Detection System

AI-powered apple disease detection using YOLOv8.

## Features
- Upload images or videos
- Real-time disease detection
- Adjustable confidence and IOU thresholds
- Downloadable results (annotated media + text reports)

## Classes Detected
- Healthy
- Black Rot
- Powdery Mildew
- Black Pox
- Anthracnose
- Codling Moth

## Dataset
- Training: 4,170 images
- Validation: 524 images
- Test: 519 images

## Model
YOLOv8s trained on custom apple disease dataset
